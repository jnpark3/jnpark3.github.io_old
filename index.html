3. Past Research
Highlighed Project:
DT-DBSCAN: Density-Based Spatial Clustering in Linear Expected Time Using Delaunay Triangulation
  • This project proposes DT-DBSCAN, a density-based clustering algorithm that can cluster datasets of size n in O(n) expected time, which is an improvement compared to existing O(n log n) density-based clustering algorithms. This improved runtime is achieved by using the Delaunay triangulation of the dataset to efficiently classify locally dense and locally sparse data points.
  • Selected to be a 2021 Regeneron Science Talent Search Scholar for this research
View Project Abstract
Other Projects:
On the Modular Properties of Hypothetical Collatz Loops
  • This project analyzes the Collatz conjecture, an unsolved problem in mathematics. A new mathematical model is created to simulate the behavior of Collatz sequences under modulo 2^n. This model is then implemented into an algorithm to prove the non-existance of Collatz loops with a length smaller than 24.
  • Selected to be an Intel International Science and Engineering Fair Finalist for this research
View Project Abstract
A Novel Approach to the Inscribed Square Problem: A Polar Graph Analysis
  • This project analyzes the inscribed square problem by observing the behavior of inscribed isosceles right triangles in Jordan curves. Although it is already known that infinitely many isosceles right triangles may be inscribed in any closed loop, this project proves that the position and orientation of all of these triangles can be represented by a continuous function. This represents a development towards a solution for the currently unsolved conjecture.
